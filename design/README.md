# App icon iterations

Simple SVG directions for the Velo Radar app icon. These avoid the red/blue contrast concern and keep the marks geometric so they should be easy to import into Icon Composer.

Open `index.html` to browse everything, or import any individual `icon-*.svg` into Icon Composer.

## Variants

1. `icon-01-teal-radar-arrow.svg` — teal radar + white navigation arrow.
2. `icon-02-mint-compass-wheel.svg` — bike-wheel/radar hybrid with a small compass cue.
3. `icon-03-graphite-dock-pulse.svg` — dock-return idea with radar rings.
4. `icon-04-cream-wheel-sweep.svg` — warm cream background, wheel + radar sweep.
5. `icon-05-forest-v-beacon.svg` — abstract V / route fork / beacon.
6. `icon-06-amber-bike-orbit.svg` — more literal bicycle, amber monochrome palette.
7. `icon-07-purple-night-radar.svg` — high-contrast night radar with a lime target.
8. `icon-08-slate-nearest-dot.svg` — minimal nearest-station radar dot.
9. `icon-09-mono-line-wheel.svg` — stark black/white wheel-radar mark.
10. `icon-10-green-dock-arrow.svg` — dock arrow + station target.
11. `icon-11-cream-center-bike-sweep.svg` — simple cream/ink radar with a centered bike.
12. `icon-12-ink-center-bike-sweep.svg` — dark version of the centered bike radar.
13. `icon-13-forest-center-bike-sweep.svg` — muted green version of the centered bike radar.
14. `icon-14-cream-target-bike-sweep.svg` — small bike replacing the target dot in the sweep.
15. `icon-15-ink-target-bike-sweep.svg` — dark version of the bike-as-target direction.
16. `icon-16-transparent-bike-radar-mark.svg` — foreground-only mark for recoloring/layering in Icon Composer.
17. `icon-17-crisp-bike-radar-light.svg` — sharper cream/ink bike with radar sweep.
18. `icon-18-crisp-bike-radar-dark.svg` — dark version of the sharper bike radar.
19. `icon-19-crisp-bike-in-sweep-light.svg` — bike as the object found inside the radar sweep.
20. `icon-20-crisp-bike-in-sweep-dark.svg` — dark version of the bike-in-sweep direction.
21. `icon-21-wheel-sweep-mark-light.svg` — more logo-like wheel/radar mark with fewer details.
22. `icon-22-wheel-sweep-mark-dark.svg` — dark version of the wheel/radar mark.
23. `icon-23-sharp-bike-target-light.svg` — larger angular bike with target rings.
24. `icon-24-transparent-crisp-bike-radar.svg` — foreground-only crisp bike/radar mark.

Notes:
- Individual icon SVGs use a 1024×1024 canvas.
- No official BIXI wordmark or logo is used.
- Most designs avoid text and fine details so they remain readable at small app-icon sizes.
- Icons 11–16 are intentionally reduced to one foreground color plus one background color, with 3–4 SVG shapes each.
- Icons 17–24 use flatter caps/joins, square SVG canvases, and more restrained cream/ink palettes for a crisper import into Icon Composer.
